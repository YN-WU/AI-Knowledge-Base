    Claude Anthropic 檔案生成

    Claude 再升級！用聊天就能生成簡報、試算表、PDF

    Claude 功能再升級。 ( 圖/翻攝自 Claude 官網 )

    Anthropic 近日正式推出 Claude 聊天機器人的檔案生成功能預覽，使用者現在不僅能編輯已上傳的 PDF、Excel、PowerPoint 等文件，還可以直接生成或轉換成相關文檔，增添文書工作處理的實用性。

    Claude 能根據您的指示創建實際文件，官方提供以下幾個應用情境參考：

    Claude 檔案生成功能應用情境

    1. 將數據轉化為分析：上傳現有文件數據給 Claude，將資料整理後，就可按需求輸出統計分析、圖表或解釋重要內容。

    2. 構建試算表：描述您的需求，Claude 就可協助創建出具備運作公式和多個表格的試算表。例如生成帶有情境分析的財務分析、建立自動化儀表板追蹤器，或帶有差異計算的模板等。

    3. 跨格式工作：Claude 可以按照你的需求處理繁瑣的資訊，並將其整理後呈現。例如可以上傳 PDF 報告轉換成 PowerPoint 簡報、分享會議記錄並整理成文件，或上傳發票並整理成帶計算功能的試算表等。

    目前這項功能僅提供給 Claude Max、Team 與 Enterprise 用戶，並計劃在未來幾周開放給 Pro 用戶。免費用戶的使用則尚未提及。

    Google Chrome Gemini

    Chrome 史上最大更新！全新 10 項 AI 功能一次看

    Chrome 瀏覽器導入全新 AI 功能。 ( 圖 / 翻攝自 Google 部落格 )

    Google 日前宣布 Chrome 瀏覽器迎來史上最大更新，導入旗下 AI 技術 Gemini 進行全面整合，以提升用戶體驗。此次更新共有 10 大亮點，包括能幫助理解網頁上的資訊，提供跨多個分頁的內容整合，也將推出代理助手功能，協助用戶完成複雜的網路任務等，讓 Chrome 變得更智能、高效又安全。

    Chrome AI 功能重點介紹

    1. 協助網頁解讀
    針對 Mac 和 Windows 的電腦用戶推出 Chrome 版的 Gemini，能幫忙解釋任何網頁或多個網頁中的複雜資訊、整理摘要重點。

    2. 代理瀏覽助手 ( 即將推出 )
    未來將在 Chrome 的 Gemini 中引入代理功能，用戶可指示 Gemini 處理複雜任務，比如執行預約、購物等重複性網路任務，過程中也隨時可以停止，確保使用者的掌控性。

    3. 多分頁內容整合
    Chrome 中的 Gemini 可在多個標籤頁中運行，協助快速比較和總結多個網站的資訊，並找到所需的內容。

    4. 查找過去的網頁瀏覽紀錄
    Chrome 中的 Gemini 可協助用戶快速找回過去曾瀏覽過的重要網站，例如可以試試看這樣下提示詞：「我上周在哪個網站看到書桌？」、「我在哪個部落格讀到了美食資訊？」，提示內容按照自己的需求改寫。

    5. 整合 Google 應用，不必切換分頁
    Gemini 與 Google 應用 ( 如日曆、YouTube 和地圖 ) 之間加深整合，用戶不必離開當前頁面就可以安排會議、查看位置詳情等。例如，如果想在 YouTube 視頻中尋找特定片段，只需詢問 Chrome 中的 Gemini， 它就會立即帶你到達該位置。

    6. 從網址列可進行 AI 模式搜尋 ( 即將推出 )
    在 Chrome 的網址列中，可快速啟用 AI Mode ( AI 模式 ) 進行搜尋。在 AI 模式中，使用者可提出更長、更複雜的問題，來獲得 AI 回覆結果，且可持續追問。

    7. 頁面即時問答
    用戶可直接在當前頁面上提出與該頁面相關的問題，Chrome 會幫你啟動搜索，提供你相對應的 AI 概述，並可以在 AI 模式中提出後續問題，進而在不離開頁面的情況下獲得有用的資訊。

    8. 加強詐騙防護
    Chrome 使用 Gemini Nano 來幫助用戶識別可能誘導下載有害軟體的技術支持詐騙，以及假病毒等網絡詐騙威脅。

    9. 減少不必要的通知干擾
    Chrome 使用 AI 技術來檢測潛在的垃圾通知或詐騙通知，並以更具體的方式提出許可請求，讓用戶選擇查看或取消訂閱，大幅減少干擾。

    10. 一步更改被破解的密碼
    Chrome 將提供 AI 功能作為密碼管理代理，能夠自動安全地填寫登錄憑證，並主動在密碼被破解時提醒用戶，支持在多個網站上，一鍵更改保存的密碼，提升帳戶安全性。

    目前，本次 Chrome 瀏覽器的更新僅開放給部分國家用戶，尚未在全球皆推出。台灣用戶若要使用還需再等等。

    阿里巴巴 Qwen3 開源模型

    阿里重磅發布 Qwen3 系列模型！ 影像、語音辨識能力大升級

    阿里雲發布 Qwen3 系列模型。( 圖 / 翻攝自通義千問 Qwen 微博 )

    阿里巴巴旗下的阿里雲日前發布了全新的 Qwen3 系列大型語言模型，包括全模態模型 Qwen3-Omni，以及多個能高品質處理視覺影像、語音、即時翻譯、圖像編輯、安全審查的模型。

    全模態模型 Qwen3-Omni

    Qwen3-Omni 為首個「原生端到端」的全模態 AI 模型，能夠接受多模態輸入，統一處理文本、圖像、音訊和影片，並透過文字和語音輸出結果，克服了過去多模態模型需要在不同能力之間取捨的困境，在多重指令下展現了出色的低延遲效果。

    Qwen3-Omni 語音識別能力甚至能堪比 Gemini 2.5-Pro 。經網友實測，Qwen3-Omni 在音訊轉字幕、影片內容理解，以及OCR（光學字元辨識）上的精準度能力也有所提升。

    大型語言模型 Qwen3-Max

    Qwen3-Max 是阿里巴巴迄今為止推出規模最大、能力最強的旗艦模型，擁有超過 1T（兆）參數，共使用 36 兆 tokens（約 48 兆字）的資料進行訓練。其在指令遵循、程式碼生成、邏輯推理和數學計算等複雜任務上表現卓越，並支援多語言與代理人（agent）應用。

    語音合成模型 Qwen3-TTS-Flash

    Qwen3-TTS-Flash 是一款不開源的收費語音合成（TTS）模型，在專業語音穩定性和音色相似度評測的表現，已超越 SeedTTS、MiniMax、GPT-4o-Audio-Preview 等主流模型。其生成的語音特色在帶有情感，聲音品質逼真。功能上支援 10 種語言，包括中、英、日、韓等多國語言，以及多種漢語方言，並可轉換 17 種不同音色。

    即時翻譯模型 Qwen3-LiveTranslate-Flash

    Qwen3-LiveTranslate-Flash 是一款高精度、多語言即時音視頻翻譯模型，支援18種語言，包括中文、英語、法語、德語等主要語言，以及普通話、粵語等方言。它最快3秒完成翻譯，速度媲美專業口譯。此外，此模型首次引入視覺上下文增強技術，就算面對噪音環境或口音較重的情況，也能透過唇語識別提升準確率。

    圖像編輯模型 Qwen-Image-Edit-2509

    Qwen-Image-Edit-2509 則擁有更高的圖像編輯一致性，支援多圖編輯，能融合處理多張圖片素材，同時內建 Control Net ，讓專業使用者能精準的控制生成效果。

    視覺語言模型 Qwen3-VL

    Qwen3-VL 具有強大的視覺處理能力。不僅可以在包含上百個物體的複雜場景中準確定位物件，並支持3D邊界框預測，精確還原物體的實際位置和大小。另外還有萬物識別的能力，範圍涵蓋名人、美食、動植物等，也內建32種語言OCR，能準確識別模糊或歪斜的文字。

    在影片理解上，Qwen3-VL 能詳細分析數小時長影片，具備進階推理能力，能分析畫面物件關係，甚至能生成對應的程式碼。

    安全審查模型 Qwen3-Guard

    Qwen3-Guard 是 Qwen 系列中首款專為安全防護設計的模型，基於強大的 Qwen3 基礎架構打造，無論是針對用戶輸入的提示，還是模型生成的回覆，都能即時檢測不當或危險內容，涵蓋 119 種語言。

    💡指令工坊

    新手入門！指令基礎架構
    「角色 + 任務 + 產出」

    掌握「角色+任務+產出」Prompt架構，與AI溝通輕鬆入門。 ( 圖 / AI 生成 )

    AI 逐漸融入每個人的生活，要和它好好交流互動，有個重要的關鍵，就是學會下指令（Prompt），讓 AI 能按照你的需求，給你想要的結果。 Prompt 要怎麼下呢？以下分享一個超簡單又實用的基礎架構：「角色 + 任務 + 產出」。

    架構解釋

    先來解釋一下這個架構：

    • 角色：當你想要得到某件事的正確答案，最穩的辦法就是找到該方面的專家來解答，而指派具體的角色設定給 AI，就像是給它戴上一頂專業帽子，讓它用對應的語氣和角度來回應。
    • 任務：有了對應的專家角色後，更重要的是就是要清楚表達「你需要什麼」，一旦目標任務明確， AI 就能更集中精神來解決你的問題。
    • 產出：當有了任務描述，可以思考你想要得到什麼形式的回答，是一段對話？一個故事？還是項目重點整理？將產出格式描述清楚， AI 更能展現更符合你需求的回答。

    情境舉例

    當你想要計畫一場省錢的旅行，套用到「角色 + 任務 + 產出」架構，可以設定為「旅遊顧問 + 省錢的旅行計劃 + 三個省錢小撇步」，接著再加上一些具體的描述，就可變成一段完整的 Prompt 如下：

    「你是一位經驗豐富的旅遊顧問，專門為旅行者提供省錢的旅遊攻略。你的任務是幫助計畫一趟省錢的旅行。請提供三個具體的省錢小撇步，幫助我在不犧牲旅遊體驗的情況下，最大限度地節約預算。」

    要留意的是，如果架構元素描述不明確，比如角色只寫了廣義的「專家」兩字、想要一趟「省錢的旅行」卻只跟 AI 說「旅行」，或是想要知道「省錢小撇步」卻只請 AI 給「建議」，最終得到的回答可能就會偏向廣泛的結果，降低實用性。

    設計 Prompt 是個需要不斷試驗和調整的過程，當你發現 AI 的回答不夠理想時，很有可能是因為指令還不夠明確和完整。只要多試幾次，你就會發現不一樣的驚喜！

    🛠️ AI 工具推薦

    NotebookLM 2025 最大更新

    Google NotebookLM 2025 年迎來大更新，不只是「做摘要」的工具，它已升級成完整的學習與知識管理 AI。輸入資料後，NotebookLM 可快速整理、分析、視覺化，並生成多媒體內容。

    主要亮點

    NotebookLM 2025 升級後，不只支援多檔案、多格式整合，還能追溯來源、生成心智圖與影音摘要，並內建 Flash Card、小測驗、報告生成等學習工具，同時打通教育平台（Canvas、Schoology，未來支援 Classroom），更全面支援學習與知識管理。

    NotebookLM 全新介面 ( 圖 / 截自NotebookLM )

    新功能操作指南

    1 Video Overview（影片摘要）

    操作步驟：點選 Video Overview → 生成影片摘要
    可自訂：段落、主題、聚焦內容
    多語言支援：包括繁體中文，影片字幕與語音可一鍵合併

    應用場景：
    • 教材或報告 → 影片化教學或研讀指南
    • 會議摘要 → 快速掌握決策
    • 專案介紹 → 提高理解度
    2 心智圖 (Mind Map)

    NotebookLM 全新功能心智圖 ( 圖/截自NotebookLM )

    操作步驟：匯入文件 → 點選心智圖 → 自動生成章節結構
    特色：節點可展開互動對話，快速掌握知識脈絡
    應用場景：報告整理、教材學習、專案規劃

    3 語音摘要 / 報告生成

    操作步驟：匯入資料 → 選擇語音摘要或報告模式 → 下指令
    特色：可自訂角色與互動模式
    應用場景：快速生成內部簡報或學習指南

    實用操作技巧

    快速整理長資料

    可匯入會議錄音、影片、文章、論壇討論或 PDF，生成摘要、重點清單、待辦事項或引用定位。
    指令示例：「整理重點與待辦，附來源」

    決策分析與結構化表格

    對於多選項或複雜決策，可生成利弊分析、考量因素與決策表。
    指令示例：「分析選項優缺點並標示風險」

    學習與知識整理

    教材、文件或筆記可生成練習題、心智圖、摘要或報告。
    指令示例：「生成練習題附答案與來源，再做心智圖」

    風險掃描與法規檢視

    對合約、條款或政策文件，可初步篩選高風險條款。
    指令示例：「找可能不利條款並標示來源」
    注意：法律決策仍需專業律師審核

    影片與語音摘要

    可將文字或影片資料生成影片摘要（Video Overview）或語音摘要，快速掌握內容。
    支援多語言（含繁體中文），可生成 MP4 影片並附字幕與語音。
    指令示例：「將此資料生成影片摘要，聚焦主要重點」
