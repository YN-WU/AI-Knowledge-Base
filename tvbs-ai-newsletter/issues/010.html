    OpenAI GPT-5 AI 模型

    GPT-5登場！新增性格設定、錯誤率大降 免費開放所有用戶

    OpenAI 推出 GPT-5。 ( 圖/翻攝自 OpenAI 官網 )

    OpenAI 日前正式發表了 GPT-5 模型，整合了 o 系列的推理能力及 GPT 系列的快速回應特點，當使用者與 GPT-5 對話時，系統會自動判斷你的意圖，來決定要用高效聊天或深度推理模型來進行回應，讓任務執行更有效率。未來 GPT-5 也將成為所有 ChatGPT 免費用戶的預設模型。

    多領域表現領先、「幻覺」發生率降低

    OpenAI 表示，GPT-5 不僅推理、處理多模態 ( 如圖文影音 ) 等各方面能力提升，在多個領域中已達最先進的水準，尤其在編程、數學、健康、視覺理解等幾大場景有明顯改進，整體也比過往的模型更少出現「幻覺」、有較低的欺騙率。經測試發現，帶思考功能的 GPT-5 虛構回應只有 4.8%，遠低於 o3 的 22% 及 GPT-4o 的 20.6%；針對健康領域測試，虛構回應僅佔 1.6%，遠低於 o3 的 15.8% 以及 GPT-4o 的 12.9%，正確率大幅提升。OpenAI 執行長奧特曼 ( Sam Altman ) 形容，和 GPT-5 對話，就像和博士級專家交談。

    新增「性格」設定，回應風格更靈活

    此外，GPT-5 推出一大特色就是新增「性格」設定，除了預設的版本 ( 輕鬆與適應力強 ) 之外，還有憤世嫉俗 ( 批評與諷刺 )、機器人 ( 高效與直率 )、傾聽者 ( 貼心並給予支援 ) 及極客 ( 有探索精神且熱情洋溢 ) 4 種性格，讓使用者不需下提示詞，就能改變不同回應風格。

    誰能用 GPT-5 ？

    所有 ChatGPT 用戶皆可免費使用 GPT-5，但是當達到一定用量後，就會自動切換成較小型的 GPT-5 mini；Plus 訂戶有更多使用量；Pro 訂戶則可無限制使用 GPT-5，且還能使用推理能力更強的 GPT-5 Pro 模型。

    價格與 API 版本說明

    GPT-5 每百萬個輸入詞元 ( token ) 為 1.25 美元、輸出詞元 10 美元，相較 GPT-4o、GPT-4.1、o3 更具成本優勢。串接 API 部分還有提供另外兩款輕量版本 GPT-5 mini 及 GPT-5 nano，GPT-5 mini 每百萬個輸入、輸出詞元分別是 0.5 和 5 美元，GPT-5 nano 則是每百萬個輸入、輸出詞元為 0.15 和 1.5 美元。

    三大AI巨頭神仙打架！OpenAI、Anthropic、Google 同時發布新模型

    Anthropic 推出 Claude Opus 4.1

    Anthropic 推出最新模型 Claude Opus 4.1。 ( 圖/翻攝自 Anthropic 官網 )

    Anthropic 搶在 OpenAI 發布 GPT-5 前夕，推出新一代 Claude Opus 4.1 模型，提升程式碼編輯、代理任務與推理能力，在用於大型語言模型 ( LLM ) 解決真實世界軟體工程問題能力的評測中獲得 74.5%，比前代進步 2 個百分點，更領先所有模型。

    根據 GitHub、樂天與 Windsurf 等企業客戶反饋，Opus 4.1 的多檔案程式重構能力提升、可在大型程式碼庫中精準定位修正，整體性能與 Sonnet 3.7 到 Sonnet 4 的升級幅度相當。目前 Opus 4.1 定價與 Opus 4 一樣，現已開放給付費訂閱用戶使用，也可在 Claude Code 中使用。

    Google Deepmind 發佈 Genie 3

    Google Deepmind 發佈世界模型 Genie 3。 ( 圖/翻攝自 Google DeepMind Youtube )

    Google DeepMind 發表全新世界模型 Genie 3，可根據文字提示即時生成互動式 3D 虛擬環境，互動時間從上一代的 10～20 秒大幅延長至數分鐘，並具備場景記憶功能，以維持畫面連貫。Genie 3 每秒可生成 24fps、720p 解析度的畫面，也可模擬真實物理現象與自然生態，支援動畫與幻想風格創作。

    此外， Genie 3 加入了「可提示的世界事件」，讓使用者透能過提示文字，即時改變生成世界中的天氣、場景、及新增物體角色，展現強大生成及理解能力。

    OpenAI 推出2款「gpt-oss」免費開源模型

    OpenAI 推出 2 款開源模型。 ( 圖/翻攝自 OpenAI X )

    OpenAI 繼 GPT-2 後，睽違六年後首度釋出開源權重語言模型 gpt-oss，主打免費下載、可自行修改與商業使用，並能在一般消費級硬體上運行。

    gpt-oss 提供兩種版本「gpt-oss-120b ( 1200 億參數 ) 」與「gpt-oss-20b ( 200 億參數 ) 」。 120b可在單一 80GB GPU 上高效運作，性能表現接近 o4-mini，適合企業應用；20b 則可在僅 16GB 記憶體的邊緣裝置 ( 例如個人筆電 ) 上執行，表現與 o3-mini 相近，適合本地推論或快速優化。

    💡指令工坊

    GPT-5 提示詞 ( Prompt ) 使用方法重點整理

    GPT-5 隆重推出。 ( 圖/翻攝自 OpenAI 官網 )

    OpenAI 推出的最新模型 GPT-5 有許多亮點，其中將所有模型統一，整合推理及非推理模型，讓自主代理能力提升更是一大特色。要怎麼樣順利使用這款模型呢？ OpenAI 官方提供了一份 GPT-5 提示技巧，幫助大家拿到更好的輸出結果。概念原則整理如下：

    1. 讓它一次做到好

    GPT-5 可以自己一步步完成任務，不用一直回來問你。

    如果想要它幫你一次完成任務，可以類似這樣說：
    「請持續進行，直到全部任務完成再結束。」
    「遇到不確定的地方，請依推理自己決定合理步驟，不用問我。」

    反之，如果想要快速得到答案，則可以嘗試說：
    「請簡短快速回答，不用深入分析。」

    2. 一句話讓它自動拆解任務

    如果你的任務描述較長，可以讓 GPT-5 自動拆成小步驟，全部做完再回報結果。
    比如可以說：「請將我的任務拆成小步驟，並逐一執行，等全部做完後再回應。」

    3. 判斷是否要看「工作過程」

    你可以決定 GPT-5 回報過程的細節程度。
    如果想要看它每一步是怎麼想的，可以說：「請邊做邊解釋原因。」
    如果不想要它回答的過程太多，則可以說 ： 「只需在最後列出關鍵步驟。」

    4. 控制回答的格式長短

    如果想看簡短的答案，可以說：「請回答精簡的重點，不需解釋。」
    如果想看長篇的答案，則可類似這樣說：「請用詳細說明，每一步都給範例。」

    5. 控制 AI 推理的深度

    GPT-5 新增 「reasoning_effort」參數，控制 AI 思考的深度與主動性，共分為低 ( low ) 、中 ( medium ) 、高 ( high ) 3 個層級，越低表示越能快速得出結論、細節越少。

    這個方法也能用在自然語言的 Prompt 中使用，比如說：「使用 high reasoning effort 的方式回答」、「使用 low reasoning effort 快速回答」。

    6. 避免模糊矛盾

    由於 GPT-5 會更嚴謹的遵循指令， Prompt 應盡量清楚，包括說明任務完成的標準、回報的方式等，不要有模糊、矛盾或重複的條件，以免導致 GPT-5 延遲回應，也確保能得到更好的回答效果。

    如果指令中有模糊的地方，可以授權 AI 自己決定，比如說：
    「你可假設已獲得資訊，若遇到模糊之處，請自行研判，不需回來問我。」
    「若遇到不明確的指示，先嘗試自行解決，待完成任務回報時再說明過程細節。」

    🛠️ AI 工具推薦

    資訊搜尋 AI 神器：Perplexity

    Perplexity AI 介面 - 結合搜尋引擎與 AI 的新一代資訊工具。 ( 圖/翻攝自 Perplexity 官網 )

    1. Perplexity 的核心技術與優勢

    Perplexity AI 是一款結合即時網路搜尋與大型語言模型的混合型 AI 工具，由前 OpenAI 研究員 Aravind Srinivas 於 2022 年創立。強項是能夠實時從網路獲取最新資訊，並透過 AI 進行整合分析，解決了傳統 AI 模型知識截止日期的限制。

    相比偏向創意型的 AI ( 如 ChatGPT 或 Google Gemini ) ，Perplexity 的獨特優勢在於：

    • 即時性：能獲取最新資訊，無知識截止日期限制。
    • 可驗證性：所有回答都附帶來源連結，提高可信度。
    • 多模型支援： Pro 版本可切換多個 AI 模型。
    • 多媒體整合：自動收集並展示相關圖片、影片與圖表。

    2. 精準搜尋的關鍵：資料來源選擇

    Perplexity 最新版本將原本的「Focus 模式」升級為更靈活的資料來源選擇系統，讓使用者能根據不同需求精確控制搜尋範圍：

    網絡

    搜索公開網站、新聞媒體與部落格，適合查詢時事、產品評價或一般知識。

    學術

    專注於學術論文、研究報告與專業期刊，適合深入研究特定學科領域。

    社交

    整合 Reddit、X 等社群平台內容，適合了解大眾觀點與即時討論。

    財經

    聚焦金融市場、股票分析與經濟報導，適合投資研究與財經資訊查詢。

    實測發現，針對不同問題選擇適合的資料來源，能顯著提升回答品質。例如查詢「NVIDIA 最新財報」時選擇「財經」，或研究「大型語言模型幻覺問題」時選擇「學術」，都能獲得更精準的結果。

    3. 使用者體驗與介面設計

    Perplexity AI 三欄式檢索結果頁面設計。 ( 圖/翻攝自 Perplexity 官網 )

    Perplexity 的介面設計極為直觀，採用三欄式布局，讓資訊呈現更有層次：

    • 左側欄： AI 生成的綜合答案，以清晰段落呈現核心資訊。
    • 中央區域：自動收集的相關圖片、影片與圖表，豐富視覺體驗。
    • 右側欄：完整引用來源列表，點擊即可查看原始內容。

    特別值得一提的是，Perplexity 引用標註系統，答案中的關鍵資訊會以灰色數字標註來源，讓使用者能輕鬆驗證內容真實性。若對生成結果不滿意，還可一鍵切換不同 AI 模型 ( 如 Claude、GPT-5、Grok 4 ) 重新生成，無需重新輸入問題。

    註冊帳號後，還能享有更多個人化功能，包括查詢歷史保存、收藏管理、對話隱私設定，以及深色模式支援，大幅提升長時間使用體驗。

    4. 進階功能與專業應用

    Perplexity Pro 訂閱版特色功能

    🔍 Pro Search

    更深入的搜尋能力，可處理複雜查詢並提供更全面的分析

    📊 Spaces

    創建專題研究空間，整合多種資料來源並生成綜合報告

    🧪 Labs

    實驗性功能，包括自動化資料整理與多模態內容分析

    Perplexity 的進階功能特別適合以下專業場景：

    • 學術研究：快速整理文獻綜述，追蹤最新研究進展。
    • 市場分析：收集競品資訊，分析行業趨勢與消費者反饋。
    • 技術開發：查詢程式碼問題，了解最佳實踐與解決方案。
    • 內容創作：收集素材，驗證事實，豐富創作背景資料。

    Perplexity AI 代表了搜尋引擎與 AI 助手的融合進化，它不僅提供答案，更提供可驗證的知識。對於需要高質量、可靠資訊的專業人士而言，它已成為不可或缺的工作工具；而對普通用戶來說，它則是一個能大幅提升資訊獲取效率的智能助手。
