    OpenAI 新模型 視覺推理

    ChatGPT 推理模型 o3 與 o4-mini 登場！首次實現「看圖思考」

    OpenAI 推出最新模型 o3 與 o4-mini。（圖／翻攝自 OpenAI）

    OpenAI 日前推出最新模型 o3 與 o4-mini，首次具備了「視覺推理」的能力，能拆解圖象中的元素，並結合如放大、裁切、旋轉等操作，進行解讀分析與判斷，也可整合 Python 腳本、網路即時搜尋、資料分析、圖片生成等功能，大幅提升解決複雜問題的能力，不再只是表面地只是看見或描述圖片而已。

    　o3 與 o4-mini 最多可支援 20 萬詞元 (tokens) 的上下文，約 15 萬字。其中，o3 屬於旗艦模型，優勢在複雜的推理與深度思考；o4-mini 性能雖然不及 o3，但速度快、成本低，適合高效率執行任務。

    　從網友實測來看，o3 不僅能辨識照片中的物體或實景，還能根據照片中的細節，推斷出拍攝地點、時間、交通資訊或活動資訊等；此外，o3 也可做到理解圖像中模糊或品質不佳的內容、辨識手寫筆記、破解迷宮等。雖然目前仍可能存在誤判或出現幻覺的錯誤，不過，據媒體與開發者的實測回饋，o3 及 o4-mini 整體表現已優於前一代模型。

    　目前 ChatGPT 免費版無法使用 o3 模型，o4-mini 有限制次數；ChatGPT Plus、Team 與 Enterprise 用戶在兩種模型上皆有限制次數；ChatGPT Pro 用戶則幾乎無限制使用。

    OpenAI 圖像生成 API

    AI 繪圖再進化！GPT-Image-1 模型上線 客製化設定創作更自由

    GPT-image-1 將四張圖片中不同的物體進行精準合成。（圖／翻攝自 OpenAI）

    OpenAI 日前推出全新圖像生成模型 GPT-image-1，並同步開放 API 供開發者透過 Playground 平台進行試用。其最大特色就是具備自訂參數的圖像調整能力，大幅提升創作自由度。

    　相較 ChatGPT 內建基礎圖像生成功能，GPT-image-1 能讓使用者自訂輸出格式，包括背景、畫質、透明度、輸出尺寸、渲染品質與壓縮格式等，皆能透過參數調整，支援輸出 JPG、PNG 或 WebP 格式，也能一次生成多張圖片，或是進行更精準的圖片局部修改與合成，提升創作效率。

    　GPT-image-1 支持文字與圖片輸入，以 token 計價，分成文字提示、圖片輸入及圖片生成 3 種計費模式，生成圖片的成本會因品質高低而有差異，讓用戶能更靈活的控制成本。

    　此模型也已被多家知名平台導入實際應用，包括 Adobe 將其整合至 Firefly 與 Express；雲端設計平台 Figma 讓使用者能透過文字提示快速生成與編輯圖像；AI 影片平台 HeyGen 運用其強化角色造型與背景的編輯能力；Canva、Instacart、Wix 等企業也在試行，探索更多創作潛力。

    Google AI影片 Veo 2

    AI 影片技術再升級！Google Veo 2 免費生成電影級影片

    Veo 2 產出效果自然的影片畫面。（圖／翻攝自 Google X）

    Google 推出最新 AI 影片生成模型 Veo 2，已於 Google AI Studio 開放免費試用，使用者只需輸入文字描述或上傳靜態圖片，就能快速生成最長達 8 秒、解析度達 720p 的生動影片，操作簡單又直覺。

    　除了基本影像生成，Veo 2 更強調模擬真實的物理效果，像是光影變化、細緻的紋理、物體互動的細節等，效果更自然逼真，宛如電影級一般。

    　至於該如何用 Veo 2 產出令人驚艷的影片？Google 官方提出了一系列建議，強調寫提示詞 (prompt) 的兩個核心原則：清晰度和細節，也就是要避免模糊不清或太過籠統的描述，而提供越多資訊，生成的影片就會越豐富、越有層次。具體優化提示詞的面向包括：

    • 主體：影片的主要焦點是什麼？
    • 動作：畫面中發生什麼事？主體是靜止、移動，還是在與某物互動？
    • 場景/背景：場景發生在哪裡？環境氛圍如何？
    • 攝影機角度/運鏡：是特寫、廣角還是動態跟拍？
    • 光線：場景的光線如何？是明亮陽光，還是昏暗帶情緒的？
    • 風格/氛圍：你想要傳達什麼樣的整體感覺或美學風格？(例如：優雅、未來感、自然寫實)

    Claude 3.7 Sonnet：寫程式與資訊視覺化的強力 AI 幫手

    AI 巨擘 Anthropic 推出的模型「Claude 3.7 Sonnet」，結合優秀的混合推理能力，成為許多專業用戶的首選。Claude AI 目前免費開放給所有用戶使用 (有每日訊息上限與流量限制)。

    Claude 3.7 Sonnet 解析：

    • 處理複雜問題：在 STEM 領域 (科學 Science、科技 Technology、工程 Engineering、數學 Math) 表現優異，尤其是數據處理、程式生成、錯誤排除。
    • 互動式內容生成：無論是有組織或凌亂的資料，都能自動建立儀表板、視覺化作品、模擬動畫。
    • 軟體開發表現：寫程式的正確率高達 62%～70%，領先業界其他模型。

    功能亮點：

    在資料視覺化領域，Claude 3.7 Sonnet 展現了令人驚豔的能力，尤其是可直接生成 SVG (可縮放向量圖形) 格式的圖表。

    例如：提供文字素材，Claude 3.7 Sonnet 就能幫你轉換成精緻的 SVG 圖表。

    2. Autopilot 更新：新增「JSON 格式化工具」

    在「Formatter」工具中，新增了「Create JSON」的功能，它可以將輸入的文字解析成可以直接使用的 JSON 物件。

    將資料轉換成 JSON 格式的好處是，能把資料從「人類可讀」變成「機器可理解和操作」，進而快速、自動化串接到各種後續流程，減少手動整理的時間和錯誤。

    使用情境舉例： 從網頁擷取到一段產品資料描述，或從表單收集到客戶回饋時，就可以轉換成 JSON 物件，接著繼續串接後面的處理步驟。
